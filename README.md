# xi.hocus
## How to (docker)
```
docker compose up --build -d
```

## How To (raw)
### Setup Node
```
nvm install 20.16.0
nvm use 20.16.0
```

### Install
```
npm i
```

### Run
```
npm run build; npm start  # windows
npm run build && npm start  # linux / mac
```

## Info
### Stack
- TypeScript
- NodeJS
- [HocusPocus](https://tiptap.dev/docs/hocuspocus) (Server)
  - Database Extension
  - Logger Extension
